Desenvolvimento de um Site para Comunidades Católicas

Estou desenvolvendo um site moderno para igrejas católicas, com o objetivo de contribuir com meu trabalho para a missão da Igreja em levar o Evangelho ao mundo. Como diz em Marcos 16:15: "Ide por todo o mundo, pregai o evangelho a toda criatura." Pensando nas realidades e dificuldades pastorais enfrentadas por muitas comunidades, decidi criar um modelo de site que permita aos fiéis conhecer melhor a história da paróquia, acompanhar notícias, acessar mídias e ficar por dentro de eventos e momentos de oração.

Cada seção do site foi pensada para atender às necessidades dos usuários (fiéis). A ideia de criar esse sistema vai muito além de simplesmente ajudar a igreja: é também uma forma de colocar em prática os conhecimentos que venho adquirindo em desenvolvimento web, testando soluções em um cenário real. O projeto está sendo desenvolvido utilizando HTML, CSS e JavaScript, com o auxílio do framework Bootstrap para o design responsivo. Para o back-end, estou utilizando PHP com MySQL como banco de dados, o que permite a criação de um painel administrativo eficiente e seguro. Além disso, estou aplicando media queries para garantir que o site funcione bem tanto em computadores quanto em dispositivos móveis.

Entre os desafios que tenho enfrentado durante o desenvolvimento deste projeto, destaco a integração entre o front-End e o back-End, a implementação da funcionalidade de edição de notícias no painel administrativo, e a lógica para o agendamento e suspensão de eventos temporários. Futuramente, pretendo migrar o banco de dados para MongoDB, integrar uma API de pagamento para seção de dízimos, criar testes automatizados para detectar e corrigir bugs e, quem sabe, hospedar o sistema em um servidor e aplicá-lo em uma igreja real.

Unir tecnologia e fé neste projeto tem sido uma experiência gratificante. Espero que este site possa, um dia, contribuir com à missão evangelizadora das comunidades católicas.

Tecnologias utilizadas

  • CSS
  • JAVASCRIPT
  • PHP
  • MYSQL

Funcionalidades do projeto

  • Galeria dos Párocos;
  • Seção de Notícias;
  • Galeria de Fotos e Videos;
  • Secretaria;
  • Agenda Semanal;
  • Contatos;

Dificuldades enfrentadas

  • Criar a opção de editar notícias no painel do administrador;
  • Integrar uma API de edição de texto no painel do administrador;
  • Criação de layout responsivo;

Aprendizados

  • Integração do Front-End com Back-End;
  • Utilização de media queries para layout responsivo;
  • Usar o JavaScript para criar animações;

Melhorias futuras
